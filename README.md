# 北太平洋海表盐度预测项目

## 项目简介

本项目基于深度学习方法，对北太平洋区域的月平均海表盐度（Sea Surface Salinity, SSS）进行预测研究。  
通过构建时空预测模型，对海表盐度数据进行建模与预测，并对不同模型的预测性能进行比较分析。

本项目主要实现：

- 海表盐度数据预处理
- 时序样本构建
- 深度学习模型训练与预测
- 不同模型性能对比
- 预测结果可视化展示

该项目同时作为本科毕业设计的研究基础。

---

## 数据来源

项目使用北太平洋区域海表盐度网格数据（NetCDF 格式）。

数据内容包括：

- 月平均海表盐度
- 时间序列数据
- 空间网格数据

数据经过以下处理流程：

1. 数据清洗
2. 缺失值处理
3. 归一化
4. 构建时间滑动窗口样本

---

## 使用模型

本项目实现并比较了以下模型：

### 1 ConvLSTM

ConvLSTM 模型能够同时捕获：

- 时间序列特征
- 空间结构特征

在本项目实验中，ConvLSTM 在保持时间序列振幅结构与相位变化方面表现最好。



### 2 CNN-TCN

CNN 用于提取空间特征  
TCN（Temporal Convolution Network）用于建模时间依赖关系。



### 3 LightGBM

LightGBM 作为传统机器学习方法进行对比实验。

---

## 实验评估指标

模型性能通过以下指标进行评估：

- MAE（平均绝对误差）
- RMSE（均方根误差）
- 相关系数（Correlation Coefficient）

ConvLSTM 在未参与训练的数据区间内仍能保持较为稳定的振幅与相位结构，在典型格点预测中相关系数最高达到0.93。
